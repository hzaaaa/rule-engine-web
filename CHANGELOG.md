# 3.0.0(2023-06-16)

### Features

- 模型数据结构重写，改由前端创建编辑并缓存模型信息，编辑完成后将模型信息一次性提交给后端
- 新增模型版本列表页面，同一个模型可以有多个版本
- 不同模型版本，新增发布状态、启用状态；发布后才可启用，同一时间最多只能启用一个版本
- 新增模型导入导出功能
